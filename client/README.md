# Store Rating Application

A full-stack web application where users can rate stores and view ratings.

## Tech Stack

### Frontend
- React.js
- Bootstrap / CSS

### Backend
- Node.js
- Express.js

### Database
- MySQL

## Features

- User Authentication
- Store Listing
- Store Rating System
- Admin Dashboard
- Role Based Access

## Installation

### Clone Repository

```bash
git clone https://github.com/faizan-shaikh-dev/store_rating.git
```

### Backend Setup

```bash
cd backend
npm install
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

## Environment Variables

Create a `.env` file in backend folder:

```env
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=yourpassword
DB_NAME=store_rating
JWT_SECRET=your_secret_key
```
